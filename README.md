# PolyCafe2025

Giao diện mẫu ứng dụng quản lý quán cà phê **PolyCafe** bằng **HTML + Bootstrap 5**.

## Cách chạy
Mở file `index.html` bằng trình duyệt hoặc chạy web server tĩnh:

```bash
python -m http.server 8000
```

Sau đó truy cập `http://localhost:8000`.

## Danh sách trang
- `index.html`: Home Dashboard
- `login.html`: Đăng nhập
- `accounts.html`: Quản lý tài khoản / phân quyền
- `change-password.html`: Đổi mật khẩu
- `employees.html`: Quản lý nhân viên
- `categories.html`: Quản lý loại sản phẩm
- `products.html`: Quản lý sản phẩm
- `cards.html`: Quản lý thẻ/bàn
- `sales.html`: Bán hàng
- `payment.html`: Thanh toán
- `bill-details.html`: Chi tiết hóa đơn
- `statistics.html`: Thống kê
- `help.html`: Trợ giúp hệ thống
